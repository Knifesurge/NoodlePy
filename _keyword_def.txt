Noodle Keyword Definitions
---------------------------------------------------------------------
set a,b
---------------------------------------------------------------------
Sets variable a to value b
---------------------------------------------------------------------
---------------------------------------------------------------------
set a, <expr>
---------------------------------------------------------------------
Sets variable a to result of expr
---------------------------------------------------------------------
---------------------------------------------------------------------
get a
---------------------------------------------------------------------
Returns the value in variable a
---------------------------------------------------------------------
---------------------------------------------------------------------
out a
---------------------------------------------------------------------
Prints the value in variable a
---------------------------------------------------------------------
---------------------------------------------------------------------
out <expr>
---------------------------------------------------------------------
Prints the result of expr
---------------------------------------------------------------------
---------------------------------------------------------------------
out;
---------------------------------------------------------------------
Pops and prints the top of the stack.
---------------------------------------------------------------------
---------------------------------------------------------------------
if <cond> then
---------------------------------------------------------------------
Starts a logical if-block
---------------------------------------------------------------------
---------------------------------------------------------------------
elif <cond> then
---------------------------------------------------------------------
Else-if part of logical if-block
---------------------------------------------------------------------
---------------------------------------------------------------------
else then
---------------------------------------------------------------------
Else part of logical if-block
---------------------------------------------------------------------
---------------------------------------------------------------------
end
---------------------------------------------------------------------
Ends a logical block
---------------------------------------------------------------------
---------------------------------------------------------------------
if <cond> startnoodling
---------------------------------------------------------------------
Starts a for loop. cond is evaluated to determine how long to loop
---------------------------------------------------------------------
---------------------------------------------------------------------
noodleuntil <cond>
---------------------------------------------------------------------
Starts a while loop. cond is evaluated every pass
---------------------------------------------------------------------
---------------------------------------------------------------------
stopnoodling
---------------------------------------------------------------------
Ends a loop block
---------------------------------------------------------------------
---------------------------------------------------------------------
